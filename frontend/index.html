Framework Methodology & Documentation

GitHub Repository

1. Preprocessing Pipeline

Enhances contrast and removes noise from woven fabric specimens using CLAHE (Contrast Limited Adaptive Histogram Equalization) followed by Gaussian Blurring and Adaptive Gaussian Thresholding to produce a clean binary mask of repeating motifs.

2. Contour Segmentation

Applies OpenCV topological contour extraction to segment boundaries of repeating design motifs. Centroids and bounding boxes are calculated for every identified component.

3. Invariant Feature Extraction

Extracts hybrid geometric features (Area, Perimeter, Circularity), texture descriptors (GLCM Contrast, Energy, Homogeneity, and Correlation), scale/rotation invariant properties (Log Hu Moments), and structural edge filters (HOG & SIFT density).

4. PCA Orientation Mapping

Applies Singular Value Decomposition (SVD) on segmented coordinates to compute principal eigenvectors aligning with the motif's primary axis, giving precise mathematical angle mapping.

PCA Centroid Localization
$$\bar{x} = \frac{1}{N}\sum_{i=1}^{N} x_i, \quad \bar{y} = \frac{1}{N}\sum_{i=1}^{N} y_i$$

Subtracts the centroid coordinates to center the contour data points.

Covariance Matrix (C)
$$C = \frac{1}{N-1} P_{\text{centered}}^T P_{\text{centered}}$$

Calculates the spatial spread of motif coordinates along 2D axes.

Eigen Decomposition & Angle
$$\det(C - \lambda I) = 0 \implies \theta = \text{arctan2}(v_{1y}, v_{1x})$$

Resolves eigenvalues (\(\lambda\)) and eigenvectors (\(v_1\)) to locate the principal orientation angle (\(\theta\)).

GLCM Homogeneity & Contrast
$$\text{Homogeneity} = \sum_{i,j} \frac{P(i,j)}{1 + (i-j)^2}$$ $$\text{Contrast} = \sum_{i,j} (i-j)^2 P(i,j)$$

Measures local micro-texture regularity and contrast variance across weavings.

This framework is developed as an intelligent system to bridge traditional handloom craftsmanship and tribal arts with modern computer vision and machine learning frameworks. By analyzing motif orientation and textures, AAM-Net enables digital archiving, weaving quality inspection, and pattern verification.
